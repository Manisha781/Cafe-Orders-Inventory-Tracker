# Café Orders & Inventory Tracker

## 📌 Project Overview
This project is a simple Python-based café management system that handles:
- Menu pricing
- Inventory tracking
- Order processing

## 🛠 Features
- Stores menu items with prices
- Maintains inventory stock
- Processes multiple customer orders
- Displays initial inventory and orders

## 💻 Technologies Used
- Python
- Dictionaries
- Lists
- Basic data structures
