# Cafe-Orders-Inventory-Tracker
This project is a simple Café Orders &amp; Inventory Tracker built using Python dictionaries and lists. It simulates a café management system where menu prices, inventory stock, and customer orders are handled efficiently. The system tracks available stock, processes multiple orders, and helps in monitoring inventory levels.
# Café Orders & Inventory Tracker

## 📌 Project Overview
This project is a simple Python-based café management system that handles:
- Menu pricing
- Inventory tracking
- Order processing

## 🛠 Features
- Stores menu items with prices
- Maintains inventory stock
- Processes multiple customer orders
- Displays initial inventory and orders

## 💻 Technologies Used
- Python
- Dictionaries
- Lists
- Basic data structures
